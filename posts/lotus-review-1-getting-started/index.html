开始学习 Lotus 源码 入门

在学习任何一个框架前，都要先问问自己：这个框架解决了什么问题？它的基础是什么？ 在 Ruby 的 web 开发世界中，我们已经拥有了 Rack，一个 ruby 应用和 web 服务器的接口和实现。 Rack 已经负责了和 web 服务器的通讯，大多数 ruby 的 web 开发框架做的事情就是从接收 http 请求开始， 分配给对应的处理器，到对于数据的验证，持久化，然后返回一个 http 响应。 这样抽象的一看，貌似也没想象中的那么可怕了。
