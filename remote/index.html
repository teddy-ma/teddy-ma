远程工作的感受

我从2016年年底开始远程工作，本文通过自问自答的形式介绍了自己的一些情况（趁着自己也还 fresh，还存留着坐班时的记忆（同理心）），希望能给对远程工作感兴趣的同学一些参考与帮助。

我是如何获得这份远程工作的？

本来我和大多数程序员一样，就职于一家互联网公司，因为家在郊区，所以通勤时间比较久，单程要两小时，本来在地铁上看书倒也不觉得浪费时间，没想到年底的时候公司突然有了新政策，不搞弹性工作制了，要求9点前必须到岗。我就算乘最早的车也来不及，不得不辞掉了工作。

既然已经离职了，那我当然打算找相对离家比较近的公司咯，不过郊区没程序员的职位，还是只能在市区找。 面试了几家后，虽然也有发了 offer 的，可是由于我还是觉得有（钱）点（不）远（够），就没接受 offer。

之后我就在网上有尝试去找 Remote 的工作，以前只是知道这种工作方式，可是自己从来没接触过，周围也没有认识的人是 Remote 的，所以也就是抱着随便看看的心态投了几份简历，没想到真的收到了回应（其中还有一次我之前不知道那家公司也是 Remote 的，电面的时候才知道，可见 Remote 也不是很少见）。

然后我就凭借着多（要）年（价）工（不）作（是）经（很）验（高）的优势，顺利入职了。

Remote 之后我的工作生活有什么样变化？

可以说是翻天覆地的变化……

吃

吃饭更健康了，虽然暂时还是叫外卖，但是自己冰箱里加点私货还是很方便的。

吃早饭时间很充裕，也吃的比较好。

晚饭有时候就自己作水果沙拉凑活了。

睡

不用担心睡过头，晚上也不会赖着不想睡刷手机，反正早睡就早起，早起了以后刷手机也是一样的。

午觉比在公司睡爽多了，毕竟是自己的床。

工作

那肯定是更加自由了，正好最近家里在装修，时刻需要家里有人，收快递啊，签收家电，物业维修等等。

这方面 Remote 的优势就体现出来了。另外半夜突然有灵感也可以直接爬起来敲代码，不用担心第二天迟到。

学习

以前看书都是首选电子书在地铁上看，现在可以在家里看书，偏向纸质书了，记笔记也用纸笔了。

另外公开课还是一样在平板上看，一般在工作的间隙也会看上几节课。

娱乐

我发现， Remote 之后我越来越少去刷论坛了，也许以前爱刷是因为觉得自己是在 带薪灌水 吧……

现在我觉得花大量时间刷论坛之类的社交网络很不值得，我一般会把这些事情放在睡前做。

我想象中的 Remote 和现实有什么不同的地方吗？

想象和现实总是有差距的，下面列举了一些我在 Remote 之前的一些 YY，以及实际的现实情况。

可以没日没夜的玩或者工作了

实际并没有，相反生活规律了很多，因为时间可以自己安排，睡得玩了那也起得晚，早睡的话第二天就早点干活，其实没什么区别。

有大把的时间可以使用

实际也还好，最大的优势就是省去了通勤的时间，以及在固定时间段工作所带来的效率的损失。

当然在沟通方面的成本也更高了，所以总体来说，工作效率并不会有太大的提高，这里顺便说一下，很多人以为 Remote 是可以提升工作效率的。

这也许有一点效果，但实际上 Remote 最大的好处就是可以让你更好的生活。

可以到处玩

玩是可以，但是要一边玩一边工作目前来看还是有点难度，等我经验更丰富一些可能可以做到。

刚开始 Remote 有什么不习惯的地方吗？

一开始肯定有的，但是过了一个礼拜基本都好了。 趁着还有印象，简单罗列一下吧：

孤独感

以前上班时都是我给别人讲段子，Remote 之后发现没有人可以说话，憋得挺难受的。 后遗症就是，现在笑点超低，看看视频弹幕就能笑趴下（后来也有所好转）。

无法进入工作状态

因为在自己家里，无法从环境上给自己暗示自己是在工作（没有同事的键盘声帮助我进入工作状态）。

坐班时的旧习惯没改掉

明明有时候白天的状态很不好，还是逼着自己坐在位置上，结果就是没有任何产出，后来想象觉得这种行为挺SB的。

那简单的做一下总结吧

我觉得 Remote 就是以增加沟通成本为代价换来了更高的个人工作效率，对于程序员和其它需要大块时间独处的职业，Remote 是一个很好的选择。
